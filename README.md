Atividade 1.a
Este programa em C cadastra alunos (nome e nota), utilizando alocação dinâmica de memória, e percorre os dados com ponteiros para identificar e exibir o aluno com a maior nota.
Atividade 1.b
O programa em C lê 5 números inteiros fornecidos pelo usuário, armazena-os em um vetor e utiliza uma estrutura de repetição para exibi-los na ordem inversa de inserção.
Atividade 1.c
O programa em C solicita a entrada de valores para uma matriz 3x3 e utiliza estruturas de repetição para percorrer seus elementos, exibindo ao final a matriz resultante com todos os valores multiplicados por 5.
Atividade 1.d
O programa em C solicita a entrada de uma matriz 3x3, gera uma matriz identidade correspondente e realiza a multiplicação entre ambas utilizando laços de repetição aninhados, exibindo a matriz identidade e o resultado da multiplicação.
Atividade 1.e
O programa em C solicita a entrada de um vetor de 3 elementos e de uma matriz 3x3, realizando a multiplicação vetorial pelas colunas da matriz por meio de estruturas de repetição, e exibindo o vetor resultante.
Atividade 1.f
O programa em C utiliza uma estrutura struct para armazenar dados de alunos e realiza o cadastro de 10 registros. Em seguida, separa os alunos em aprovados e reprovados com base na média final, armazenando-os em vetores distintos e exibindo os resultados.
Atividade 1.g
O programa em C utiliza uma estrutura struct para armazenar dados de livros e realiza o cadastro de 5 registros. Em seguida, implementa uma busca sequencial por título utilizando a função strcmp, exibindo os dados do livro caso ele seja encontrado.
Atividade 1.h
O programa em C realiza a leitura de 5 valores inteiros armazenados em um vetor e utiliza ponteiros para acessar os elementos da memória, exibindo o dobro de cada valor informado.
Atividade 1.i
O programa em C implementa uma função que utiliza ponteiros para ordenar três valores inteiros em ordem crescente. A função também verifica se os valores são iguais, retornando um indicador booleano, e exibe os resultados ao final.
Atividade 1.j
O programa em C utiliza alocação dinâmica de memória com malloc para armazenar um conjunto de alunos. Em seguida, percorre os registros utilizando ponteiros para identificar o aluno com a maior nota e exibir suas informações.
Atividade 2.a
O programa em C implementa uma estrutura de pilha utilizando listas encadeadas para verificar o balanceamento de uma expressão. Ele analisa caracteres de abertura e fechamento de símbolos como parênteses, colchetes e chaves, garantindo a correta correspondência entre eles.
Atividade 2.b
O programa em C implementa uma pilha dinâmica utilizando lista encadeada para inverter uma string. Os caracteres são empilhados e depois desempilhados seguindo o princípio LIFO (Last In, First Out), resultando na string invertida.
Atividade 3.a
O programa em C implementa uma estrutura de fila dinâmica utilizando lista encadeada para simular um sistema de atendimento. Cada cliente possui um tempo de atendimento, e o sistema calcula o tempo de espera acumulado conforme a ordem de chegada (FIFO).
Atividade 3.b
O programa em C implementa uma fila de impressão com prioridade utilizando lista encadeada. Os documentos são inseridos de forma ordenada conforme o nível de prioridade e processados seguindo essa ordem, simulando um sistema de gerenciamento de impressão.
